
## Exo RPG

### Créer une classe "Race"

- name : string


### Créer la classe "Hero"

- name : string
- race : Race
- level : int
- hp : int
- hpMax : int 
- mana : int
- manaMax : int
- strength : int
- agility : int
- intelligence : int
- defense : float
- scoreCriticalStrike : int
- criticalDamage : int
- damageMin : int
- damageMax : int


* Par défaut, tous les héros créés sont de niveau 1
* scoreCriticalStrike : 12%
* criticalDamage : 150%
